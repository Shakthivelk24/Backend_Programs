# PROGRAM 01
### This repo holds a basic web app setup using HTML, CSS, and JS. It also has a small server entry script.

# Install Dependencies(Run in VS Code Terminal)

### Step 1
```
npm init -y
```
### Step 2
```
npm install express
```

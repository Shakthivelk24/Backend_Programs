# To Clone My Repository 

### Step 1:
```
git clone https://github.com/Shakthivelk24/Backend_Programs.git
```
### Step 2:
```
cd Pro(1,2A,2B,2C,3A,3B)
npm install
```
### Step 3
Change the MongoDB 
```
mongodb+srv://<username>:<password>@cluster0.xxxxxx.mongodb.net/
```

### To Start the Server 
```
npx nodemon app.mjs
```
